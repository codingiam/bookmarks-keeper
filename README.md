# 📚 Bookmarks Keeper

**Bookmarks Keeper** is a lightweight Ruby web app for saving, organizing, and searching your favorite links.
Instead of relying on scattered browser bookmarks, this tool helps you keep all your favorite resources in one place, easily searchable and structured.

Browser bookmarks are convenient, but they quickly become messy and hard to manage:
- Different browsers don’t sync well across devices.
- Organizing links into folders is limiting.
- Searching through long lists of bookmarks can be frustrating.

**Bookmarks Keeper** solves this by giving you a centralized, searchable, and customizable space for your links. Instead of fighting with browser bookmark managers, you get a dedicated web app designed specifically for collecting and organizing resources.
